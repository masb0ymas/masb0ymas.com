### Description
Moneyflow ID is a comprehensive web application designed to help you take control of your financial life by tracking and managing your cashflow effectively. The platform offers intuitive tools for monitoring income sources, categorizing expenses, creating personalized budgets, generating financial reports, and setting savings goals to improve your overall financial health and decision-making.

### Features
- Income
- Expense
- Budgeting
- Report
- Savings

### Tech Stack

- Mantine UI
- TypeScript
- Zustand (State Management)
- React Query
